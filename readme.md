URL: (準備中)


## 概要
---
本アプリでは手軽に蔵書管理ができることに加え、書籍ごとにメモを保存することができます。
メモはフォルダーごとに管理できますので、引用文(アプリ名であるQuote)や語彙フォルダーを作って記録したり、
読書中の疑問やメモに使うなど、利用者の用途に合わせて自由な使い方ができます。
